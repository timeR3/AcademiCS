# **App Name**: IntelliLearn

## Core Features:

- User Authentication: User authentication and role management (teacher/student) to control access to features.
- Syllabus Generation: AI-powered tool for syllabus generation based on teacher-provided content, using createSyllabus action.
- Syllabus Management: Accept, generate new, and save syllabus features.
- Questionnaire Generation: AI tool for automatically generating questionnaires based on provided content.
- Course Progress Visualization: Display course levels with clear visual indicators (locked, in progress, completed).
- Evaluation Interface: Clean and distraction-free evaluation interface, including result modal.
- Centralized API Communication: A single reusable API function to communicate with the backend webhook for all data transactions.

## Style Guidelines:

- Primary color: Teal (#008080), echoing themes of trust and stability. The visual associations of teal align with reliability.
- Background color: Off-white (#F5F5F5), to give a calm, clean appearance and allow the teal components to stand out. Chosen to avoid stark white which can be too bright for educational environments.
- Accent color: Cyan (#00FFFF), to add contrast. Since teal evokes precision and rigor, this pairing complements the visual and supports learning and insight.
- Body: 'PT Sans' (sans-serif) because it's humanist and readable, so it's good for an educational product
- Headline: 'Playfair' (serif), matched with PT Sans.
- Use clear and intuitive icons to represent different course levels, evaluation statuses, and actions.
- Subtle transition animations and micro-interactions to provide feedback.
